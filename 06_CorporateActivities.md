# 6-3. 会計・財務

## 会計

```
損益分岐点 = 固定費 + 変動費
          = 固定費 / (1 - 変動費率)
```

```
限界利益率 = 限界利益 / 売上高
          = (売上高 - 変動費) / 売上高
```

## 連結決算

- 議決権の過半数を実質的に保有している会社
- 上記でなくても、役員を派遣し意思決定機関を支配していること

## 経済的付加価値(EVA：Economic Value Added)
税引き後営業利益から資本費用を差し引いた余剰利益

```
EVA = 税引き後営業利益 - 資本費用
```
